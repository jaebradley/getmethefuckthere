### Install
`npm install getmethefuckthere -g`

### Usage
* `gmtft -o 'origin' -d 'destination'`
  * When no travel mode is specified, then directions default to `driving`
* `gmtft -o 'origin' -d 'destination' -t 'travel mode'`
  * Travel Modes
    * `driving`
    * `walking`
    * `bicycling`
    * `transit`
